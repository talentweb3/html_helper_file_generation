Hinweisdatenbanken verschmelzen

Sie können  Hinweisdatenbanken verschmelzen, indem Sie das  Datenbanken verschmelzen... Kommando vom Menü im  Datenbank Manager wählen. Wählen Sie einfach eine Datenbank, deren Hinweise sie zur aktuell ausgewählten hinzufügen wollen.  

Falls der Crossword Compiler die Datenbank, die sie verschmelzen wollen nicht erstellt hat, sollten Sie sicherstellen, dass die Datenbank im  CSV (Comma Separated Values) Format gespeichert ist. MS Excel, und viele andere Arbeitsprogramme können im  CSV Format speichern.
