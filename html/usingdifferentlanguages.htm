Andere Sprachen verwenden

Crossword Compiler ist in der Lage, Rätsel in vielen verschiedenen Sprachen zu machen und unterstützt den ganzen Unicode Zeichensatz.

Spracheinstellungen ändern

Ändern Sie die Worte, die anstatt "waagerecht", "senkrecht", etc. verwendet werden sollen unter Hinweiseigenschaften. Sie können auch andere Einstellungen auf der sprachspezifische Einstellungen Seite in den Hinweiseigenschaften ändern. Speichern Sie die Datei als Vorlage, um Ihre Einstellungen beim nächsten Mal, wenn Sie ein Rätsel machen, wieder zu verwenden.

Interaktive Java Rätsel exportieren

Übersetzen Sie die Untertitel der Knöpfe in der Java Lösungsanwendung unter Anwendungseigenschaften. Sie können auch die Abschlussbotschaft ändern.

Fremdsprachige Zeichen nutzen

Es gibt eine Option auf der Knöpfe Seite der Anwendungseigenschaften, um "Immer Alphabet anzeigen" nutzen zu können. Dies zeigt eine Liste von Buchstaben unter dem Rätsel, sodass der Lösende auf diese klicken kann, um Zeichen in das Rätsel ein zu setzen. Dies erlaubt auch Personen mit jeder Computereinstellung, mehrsprachige Rätsel einfach durch Klicken mit der Maus zu lösen, auch wenn sie die Zeichen auf ihrer Tastatur nicht haben. Sie angezeigten Zeichen im Alphabet werden unter Hinweiseigenschaften, Sprachbesonderheiten, eingestellt.

Webseiten exportieren

Der Webseiten Export unterstützt die meisten Sprachen. Sie können Ihre eigenen HTML Vorlagen erstellen oder in jede Sprache die Sie möchten übersetzen. Schauen Sie imHTML VorlagenThema nach Details.

Verschiedene Zeichensätze in Wortlisten nutzen

Crossword Compiler Wortlisten unterstützen verschiedene Zeichensätze entweder voller Unicode oder eigene. Schauen Sie im Zeichensätze Thema nach mehr Informationen.
