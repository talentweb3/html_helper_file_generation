Dateien Sicherheitskopieren und wiederherstellen

Benutzen Sie die Menüpunkte am unteren Rand des Optionen Menüs, um Sicherheitskopien zu machen oder alle Dateien Ihrer Crossword Compiler Wortlisten, Datenbanken, Vorlagen und Einstellungsdateien wiederherzustellen. Seien Sie sich bewusst, dass diese Option keine Sicherung Ihrer Rätseldateien macht:. Diese sollten separat gesichert werden, beispielsweise als generelle Sicherheitskopie Ihres "Eigene Dateien" Ordners.

Benutzen Sie Dateien und Einstellungen Sicherheitskopieren im Optionen Menü, um alle Ihre Dateien in eine einzige ‘cab’ Datei zu sichern, welche später benutzt werden kann, falls Sie ein Problem haben. Sie werden nach einem Verzeichnis gefragt und einem Dateinamen, unter dem Sie Ihre Sicherheitskopien ablegen wollen. Die Datei könnte so groß sein, dass sie nicht auf eine floppy disk passt. Es mag nützlich sein, Ihre Dateien auf Ihrer Festplatte zu speichern, und sie danach auf eine CD-R oder DVD-R oder auf einem Web-Service zu speichern. 

Um Ihre Dateien und Eintsellungen von einem Backup wieder her zu stellen, verwenden Sie Zurücksetzen von Sicherheitskopie aus dem Menü Optionen. Wählen Sie die gewünschte Backupdatei und starten Sie den Crossword Compiler neu. Beachten Sie, dass alle vorhandenen Dateien und Einstellungen durch die des Backups ersetzt werden.

Sie können diese Option auch nutzen, um Einstellungen leichter auf einen neuen Computer zu übertragen. Installieren Sie Crossword Compiler einfach auf dem neuen Computer, benutzen Zurücksetzen von Sicherheitskopie, um die Standardeinstellungen und Dateien zu ersetzen. Danach löschen Sie den Crossword Compiler vom alten Computer.

Zurücksetzen einzelner  Wortlisten

Sie können Wortlisten auch sichern und zurücksetzen, indem Sie die Option im Dateien Menü des Wortlisten Manager nutzen. Die Option Wiederherstellen von Sicherheitskopie kann benutzt werden, um eine oder mehrere selektierte Wortlisten wiederherzustellen. Die Sicherheitsdatei kann entweder nur aus Wortlisten bestehen - durch Benutzung von  Alle Wortlisten Sicherheitskopieren - oder eine der vollen Sicherheitsdateien sein, wie oben beschrieben.
