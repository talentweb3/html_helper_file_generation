Tipp Botschaften

Crossword Compiler wird Tipp Botschaften für einige Funktionen hervorbringen, um Ihnen zu helfen, wenn Sie diese die ersten paar Male verwenden. Sie können die Tipp Botschaften zukünftig überspringen, indem Sie auf den Zeige mir das nicht nochmal Knopf klicken. Es gibt eine Reihe ähnlicher Optionen zum Überspringen von Schritten unter Neues Rätsel.

Sie können alle Tipps die sie übersprungen haben wieder erlauben, indem Sie Alle Tipp Botschaften erlauben im  Optionen Menü klicken.
