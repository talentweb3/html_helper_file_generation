Hinweise importieren

Wählen Sie  Hinweise importieren im Hinweis Menü, um Hinweise aus einer einfachen Textdatei einzufügen. Sie sollten das Gitter bereits gefüllt haben, bevor Sie diese Funktion nutzen. Wenn Sie eine einfache Textdatei mit Gitterdaten importieren wollen, nutzen Sie Öffnen im Datei Menü.

Der Import erwartet von der Datei, waagerechte Hinweise gefolgt von senkrechten Hinweisen zu enthalten. Es mag eine Hinweisüberschrift  enthalten und eine Hinweisnummerierung wird, falls vorhanden, ignoriert.

Nachdem der Import abgeschlossen ist, wird das Hinweise prüfen Fenster geöffnet, damit Sie prüfen können, ob alles korrekt importiert wurde.
