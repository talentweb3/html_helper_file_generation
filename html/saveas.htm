Speichern unter

Sie können einem Rätsel einen Namen geben, oder es unter einem neuen Namen speichern, indem Sie Speichern unter... im Datei Menü wählen. Geben Sie einen Namen für das Rätsel ein und drücken Sie OK. Sie können einen anderen Ordner wählen oder eine andere Festplatte zum Speichern, wie Sie mögen.

Sie können das Rätsel mit dem aktuellen Namen speichern, indem Sie Ctrl + S drücken, oder Speichern im Datei Menü wählen.

Um im alten Crossword Compiler 4/5 Format zu speichern, wählen Sie die Option aus dem Speichern als Typ von der drop-down Box. Anderenfalls benötigen die Dateien Crossword Compiler 6 oder höher, um geöffnet zu werden.

Um Rätsel zur Nutzung mit anderen Programmen zu speichern, nutzen Sie die Export Funktion.
