WordWeb

WordWeb ist ein englischer Thesaurus/Wörterbuch Zusatz für Crossword Compiler. Nachdem Sie ihn installiert haben, können Sie ihn aus dem Hinweis Editor oder vom WorteMenü verwenden, um ausgewählte Worte nachzuschlagen. Um ein Wort (oder ein Buchstabenmuster) im Rätsel nach zu schlagen, drücken Sie einfach Ctrl + W. Dies kann verwendet werden, um automatische Antworten korrekt zu formatieren. (wenn diese Option unter Präferenzen ausgewählt worden ist).

Für mehr Information zur Verwendung von WordWeb wählen Sie Hilfe vom Menü des WordWeb Fensters.

Information über WordWeb ist auch im Internet verfügbar unter https://www.crossword-compiler.com/wordweb.html.
