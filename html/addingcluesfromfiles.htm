Hinweise aus Dateien einfügen

Wählen Sie Hinweise aus Dateien einfügen mit dem Menü Knopf im  Hinweisdatenbank Manager  um alle Hinweise aus einem Set von Dateien zu einer Hinweisdatenbank hinzuzufügen.

Wählen Sie die Dateien deren Hinweise Sie hinfügen wollen (halten Sie Ctrl gedrückt und klicken auf jede Datei) und drücken Sie  Öffnen. Sie werden dann gefragt, ob Sie die Hinweise mit Datum der Datei, aus der sie kommen, markieren wollen. Wenn Sie JA  clicken, werden die Hinweise nach Datum sortiert in die Datenbank aufgenommen und Sie werden das Datum später im Hinweis Editor sehen können, wenn der Hinweis angezeigt wird.
