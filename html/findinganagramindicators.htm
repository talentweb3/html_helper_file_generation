Anagramm Indikatoren

Anagramm Indikatoren werden in kryptischen Rätseln zur Anzeige von Anagrammen benutzt. Um eine Liste von Indikatoren nach Anfangsbuchstabe sortiert an zu schauen,, drücken Sie Ctrl+G im Hinweis Editor. Ein Menü wird erscheinen und Sie können einen Indikator in den Hinweis einfügen, indem Sie das Suchwortmenü wählen. Falls Sie es wünschen, können Sie die Liste anpassen. Schauen Sie unter dem Nachschlagelisten Thema nach weiteren Informationen.
